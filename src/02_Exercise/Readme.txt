OpenCL installation

1. install Intel OpenCL SDK: https://software.intel.com/content/www/us/en/develop/tools/opencl-sdk.html
2. install Intel OpenCL runtime: https://software.intel.com/content/www/us/en/develop/articles/opencl-drivers.html
3. copy build customization files from C:\Program Files (x86)\IntelSWTools\system_studio_2020\OpenCL\sdk\BuildCustomizations 
                                    to C:\Program Files (x86)\Microsoft Visual Studio\2019\Enterprise\MSBuild\Microsoft\VC\v160\BuildCustomizations
